# sql-cte-visualizer

次の記事のコードを動かしただけ。

> [SQL スクリプト上でのテーブルや共通テーブル式の依存関係を可視化する｜ Dentsu Digital Tech Blog](https://note.com/dd_techblog/n/n3876f38cc5fc)

## 環境構築

venv 等で適当に Python 環境を用意して `pip install -r requirements.txt` すれば OK。

## 使い方

```
$ streamlit run main.py
```
